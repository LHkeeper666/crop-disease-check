# 病虫害学家技能知识库

> 本文件为农作物疾病检测系统 Agent 的「病虫害学家」专业技能知识库，涵盖病虫害识别、防治方案、农药使用规范、综合防治策略等领域。
> 病害数据对应 YOLOv8 病害模型 38 个 class，虫害数据对应 102 个 class。

## 一、病害识别与防治

### 1.1 真菌性病害

#### 番茄早疫病（Alternaria solani）
- **症状**：叶片出现同心轮纹状病斑，边缘黄色晕圈；茎部椭圆形褐色凹斑；果实蒂部黑色凹陷
- **发病条件**：温度24-28℃，高湿交替，连作地块易发
- **防治方案**：
  - 农业防治：轮作3年以上，清除病残体，合理密植
  - 化学防治：发病初期喷施代森锰锌600倍液，或嘧菌酯1500倍液，7-10天一次
  - 生物防治：木霉菌制剂灌根

#### 番茄晚疫病（Phytophthora infestans）
- **症状**：叶片水渍状暗绿病斑，湿度大时叶背白色霉层；茎部黑褐色条斑；果实褐色硬斑
- **发病条件**：温度15-25℃，相对湿度>75%，冷凉潮湿
- **防治方案**：
  - 农业防治：选用抗病品种，及时通风降湿，避免大水漫灌
  - 化学防治：甲霜灵锰锌600倍液，或烯酰吗啉1500倍液
  - 关键：发现中心病株立即拔除，周围植株喷药保护

#### 番茄叶霉病（Fulvia fulva）
- **症状**：叶片正面黄色斑，背面灰紫色霉层；严重时叶片卷曲干枯
- **发病条件**：温度20-25℃，相对湿度>80%
- **防治方案**：
  - 农业防治：增加通风，降低种植密度
  - 化学防治：多菌灵500倍液，或苯醚甲环唑2000倍液

#### 番茄斑枯病（Septoria lycopersici）
- **症状**：叶片出现大量灰白色小斑点，中心有黑色小点（分生孢子器）
- **发病条件**：温度20-25℃，温暖多雨
- **防治方案**：苯醚甲环唑2000倍液，清除病叶

#### 番茄靶斑病（Corynespora cassiicola）
- **症状**：叶片出现靶形病斑，同心轮纹明显
- **发病条件**：温度20-28℃，高湿环境
- **防治方案**：嘧菌酯1500倍液，增加通风

#### 辣椒细菌性斑点病（Xanthomonas）
- **症状**：叶片出现水渍状小斑点，后期褐色坏死；果实隆起斑点
- **发病条件**：温度25-30℃，温暖多雨，暴风雨后易爆发
- **防治方案**：
  - 农业防治：轮作，选用无病种子
  - 化学防治：铜制剂（氢氧化铜）600倍液，或噻唑锌800倍液

#### 马铃薯早疫病（Alternaria solani）
- **症状**：叶片同心轮纹病斑，与番茄早疫病类似
- **防治方案**：代森锰锌600倍液，轮作

#### 马铃薯晚疫病（Phytophthora infestans）
- **症状**：叶片水渍状病斑，叶背白色霉层；块茎褐色干腐
- **防治方案**：甲霜灵锰锌，及时通风，切刀消毒

#### 葡萄黑腐病（Guignardia bidwellii）
- **症状**：果实变黑皱缩，叶片褐色斑点
- **防治方案**：清除病果，喷施波尔多液

#### 葡萄霜霉病（Plasmopara viticola）
- **症状**：叶片正面黄色斑，背面白色霜霉层
- **防治方案**：烯酰吗啉，改善通风

### 1.2 病毒病

#### 番茄黄化曲叶病毒病（TYLCV）
- **症状**：新叶黄化卷曲，植株矮化，果实小而少
- **传播途径**：烟粉虱传播
- **防治方案**：
  - 核心：防治烟粉虱（吡虫啉、噻虫嗪）
  - 选用抗病品种（Ty系列抗性基因）
  - 发现病株立即拔除

#### 番茄花叶病毒病（TMV）
- **症状**：叶片花叶、皱缩、畸形
- **传播途径**：接触传播、种子带毒
- **防治方案**：
  - 种子消毒（10%磷酸三钠浸种20分钟）
  - 轮作，拔除病株
  - 操作前手消毒（肥皂水洗手）

### 1.3 细菌性病害

#### 桃细菌性穿孔病（Xanthomonas arboricola）
- **症状**：叶片出现水渍状小斑，后期脱落形成穿孔
- **防治方案**：农用链霉素，清除落叶

#### 柑橘黄龙病（Candidatus Liberibacter）
- **症状**：叶片斑驳黄化，果实畸形着色不均
- **传播途径**：柑橘木虱传播
- **防治方案**：
  - 核心：防治木虱（吡虫啉、噻虫嗪）
  - 发现病株立即挖除销毁
  - 建立无病苗圃

## 二、虫害识别与防治

### 2.1 刺吸式害虫

#### 蚜虫类
- **危害**：吸食汁液，分泌蜜露引发煤污病，传播病毒
- **识别**：群集于嫩梢、叶背，体小色绿/黑/红
- **防治**：
  - 黄板诱杀
  - 释放天敌（瓢虫、草蛉、食蚜蝇）
  - 化学：吡虫啉2000倍液、啶虫脒1500倍液

#### 白粉虱
- **危害**：吸食汁液，分泌蜜露，传播病毒
- **识别**：叶背白色小飞虫，受惊起飞
- **防治**：
  - 黄板诱杀
  - 释放丽蚜小蜂
  - 化学：噻虫嗪2000倍液、联苯菊酯1500倍液

#### 烟粉虱
- **危害**：传播TYLCV等病毒
- **识别**：比白粉虱更小，体色浅
- **防治**：同白粉虱，重点防治以控制病毒病

#### 红蜘蛛（叶螨）
- **危害**：吸食叶片汁液，叶片失绿、干枯
- **识别**：叶背红色小螨，结细网
- **防治**：
  - 增加湿度（螨类喜干燥）
  - 释放捕食螨（智利小植绥螨）
  - 化学：阿维菌素2000倍液、螺螨酯3000倍液

#### 褐飞虱
- **危害**：吸食水稻汁液，传播病毒
- **识别**：成虫褐色，群集稻丛基部
- **防治**：保护天敌（蜘蛛），喷施吡蚜酮

### 2.2 食叶害虫

#### 菜青虫（菜粉蝶幼虫）
- **危害**：取食十字花科蔬菜叶片
- **识别**：绿色毛虫，体侧有黄线
- **防治**：
  - 人工捕捉
  - 苏云金杆菌(Bt)制剂
  - 化学：甲维盐2000倍液

#### 小菜蛾
- **危害**：十字花科蔬菜主要害虫
- **识别**：幼虫纺锤形，受惊扭动后退
- **防治**：Bt制剂+甲维盐轮换使用

#### 稻纵卷叶螟
- **危害**：幼虫卷叶取食，影响水稻光合作用
- **识别**：幼虫绿色，纵卷叶片成筒状
- **防治**：释放赤眼蜂，喷施甲维盐

### 2.3 钻蛀性害虫

#### 亚洲玉米螟
- **危害**：幼虫蛀食玉米茎秆和果穗
- **识别**：茎秆有排粪孔，果穗被蛀食
- **防治**：
  - 释放赤眼蜂
  - Bt制剂灌心
  - 化学：氯虫苯甲酰胺2000倍液

#### 三化螟
- **危害**：幼虫蛀食水稻茎秆造成枯心
- **防治**：灯光诱杀，喷施杀螟丹

### 2.4 地下害虫

#### 蛴螬（金龟子幼虫）
- **危害**：咬食作物根系
- **识别**：白色C形虫体，生活在土中
- **防治**：土壤处理（辛硫磷颗粒），灯光诱杀成虫

#### 蝼蛄
- **危害**：咬食种子和根系，拱土造隧道
- **防治**：毒饵诱杀（麦麸+辛硫磷），撒施辛硫磷颗粒

#### 金针虫（叩头虫幼虫）
- **危害**：咬食种子和幼苗根部
- **防治**：土壤处理（毒死蜱颗粒）

#### 小地老虎
- **危害**：幼虫咬断幼苗茎基部
- **识别**：灰褐色幼虫，体表粗糙
- **防治**：毒饵诱杀，喷施辛硫磷

## 三、农药使用规范

### 3.1 农药分类

| 类别 | 代表药剂 | 特点 | 注意事项 |
|------|----------|------|----------|
| 有机磷 | 辛硫磷、毒死蜱 | 广谱，触杀+胃毒 | 毒性较高，注意安全 |
| 氨基甲酸酯 | 吡蚜酮、克百威 | 选择性好 | 对鱼类有毒 |
| 拟除虫菊酯 | 溴氰菊酯、高效氯氟氰菊酯 | 高效，击倒快 | 对蜜蜂高毒 |
| 新烟碱类 | 吡虫啉、噻虫嗪 | 内吸传导，持效期长 | 对蜜蜂有风险 |
| 阿维菌素类 | 阿维菌素、甲维盐 | 生物源，低毒 | 对螨类、线虫有效 |
| 双酰胺类 | 氯虫苯甲酰胺 | 新型，对鳞翅目高效 | 安全性好 |

### 3.2 安全间隔期（重要！）

| 作物 | 药剂 | 安全间隔期 |
|------|------|------------|
| 番茄 | 吡虫啉 | 7天 |
| 番茄 | 阿维菌素 | 7天 |
| 番茄 | 百菌清 | 7天 |
| 黄瓜 | 多菌灵 | 5天 |
| 黄瓜 | 代森锰锌 | 15天 |
| 辣椒 | 氯氰菊酯 | 7天 |
| 水稻 | 三唑磷 | 21天 |
| 水稻 | 吡蚜酮 | 14天 |

### 3.3 用药原则

1. **对症用药**：根据病虫害种类选择药剂，不盲目加大用量
2. **交替用药**：不同作用机理药剂轮换使用，延缓抗药性
3. **适时用药**：害虫低龄期、病害初期用药效果最好
4. **安全用药**：遵守安全间隔期，注意防护
5. **精准用药**：使用推荐浓度，不随意加大用量

### 3.4 抗药性管理

- 轮换使用不同作用机理的药剂
- 避免同一药剂连续使用超过3次
- 使用复配制剂（但要注意合理复配）
- 结合非化学防治手段

## 四、综合防治策略（IPM）

### 4.1 防治层次

```
第一层：农业防治（基础）
    ↓
第二层：物理防治（辅助）
    ↓
第三层：生物防治（核心）
    ↓
第四层：化学防治（应急）
```

### 4.2 农业防治

- 选用抗病品种
- 合理轮作倒茬
- 清洁田园，清除病残体
- 合理肥水管理，增强植株抗性
- 适时播种，避开病虫害高发期

### 4.3 物理防治

- **黄板**：诱杀蚜虫、白粉虱、斑潜蝇（每亩30-40块）
- **蓝板**：诱杀蓟马（每亩20-30块）
- **灯光诱杀**：黑光灯、频振式杀虫灯
- **防虫网**：40-60目防虫网阻隔害虫
- **银灰色地膜**：驱避蚜虫

### 4.4 生物防治

| 天敌/制剂 | 防治对象 | 使用方法 |
|-----------|----------|----------|
| 赤眼蜂 | 玉米螟、卷叶螟 | 每亩释放1-2万头 |
| 瓢虫 | 蚜虫 | 每亩释放200-500头 |
| 草蛉 | 蚜虫、粉虱 | 每亩释放500-1000头 |
| 捕食螨 | 红蜘蛛 | 每亩释放10-20万头 |
| Bt制剂 | 鳞翅目幼虫 | 500-1000倍液喷雾 |
| 木霉菌 | 土传病害 | 灌根处理 |
| 枯草芽孢杆菌 | 多种病害 | 喷雾或灌根 |

## 五、本系统检测模型覆盖范围

### 5.1 病害模型（38类）

覆盖作物：苹果、蓝莓、樱桃、玉米、葡萄、柑橘、桃、辣椒、马铃薯、树莓、大豆、南瓜、草莓、番茄

主要病害类型：
- 真菌性病害：早疫病、晚疫病、白粉病、黑腐病、锈病、叶霉病等
- 细菌性病害：细菌性斑点病、穿孔病
- 病毒病：花叶病毒、黄化曲叶病毒
- 生理性病害：健康状态识别

### 5.2 虫害模型（102类）

覆盖害虫类型：
- 鳞翅目：螟虫、夜蛾、菜青虫、小菜蛾等
- 同翅目：飞虱、蚜虫、叶蝉等
- 鞘翅目：金龟子、象甲、叩头虫等
- 缨翅目：蓟马
- 蜱螨目：红蜘蛛、锈螨等
- 地下害虫：蛴螬、蝼蛄、金针虫、地老虎等

## 六、应急处理流程

### 发现疑似病虫害时的处理步骤：

1. **拍照记录**：拍摄病虫害全景和特写照片
2. **初步判断**：根据症状描述初步判断类型
3. **使用系统检测**：上传图片至图像检测模块，获取AI识别结果
4. **确认诊断**：结合AI结果和专业知识确认
5. **制定方案**：根据病虫害类型和严重程度制定防治方案
6. **执行防治**：按照方案执行，注意安全间隔期
7. **跟踪效果**：防治后3-7天检查效果，必要时补防

### 紧急情况处理：

- **爆发性害虫**：立即使用速效药剂控制，如高效氯氟氰菊酯
- **流行性病害**：清除病株+周围保护性喷药
- **新发病虫害**：取样送检，先隔离后处理
