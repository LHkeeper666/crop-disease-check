## 一、作物种植管理

### 1.1 种植季节与茬口安排

### 1.3 轮作原则

**轮作的好处**：
- 打破病虫害生活史循环，减少土传病害
- 平衡土壤养分消耗
- 改善土壤微生物群落结构
- 减少杂草种子库积累

**推荐轮作方案**：
- 番茄 → 豆类 → 白菜类（3年轮作）
- 黄瓜 → 葱蒜类 → 茄果类
- 玉米 → 大豆 → 小麦（粮食轮作）
- 水稻 → 油菜 → 水稻（水旱轮作）

**忌连作作物**：番茄、黄瓜、辣椒、西瓜、大豆（连作障碍明显）

## 二、施肥方案

### 2.1 肥料三要素

| 要素 | 作用 | 缺乏症状 | 过量症状 |
|------|------|----------|----------|
| 氮(N) | 促进茎叶生长，叶绿素合成 | 叶片发黄，植株矮小 | 徒长，抗性下降，延迟成熟 |
| 磷(P) | 促进根系发育，花果形成 | 叶片暗绿/紫红，根系弱 | 早熟，籽粒不饱满 |
| 钾(K) | 增强抗逆性，促进果实膨大 | 叶缘焦枯，茎秆软弱 | 影响钙镁吸收 |

### 2.2 不同作物施肥方案

#### 番茄施肥方案（亩产5000kg目标）
- **基肥**：腐熟有机肥3000-5000kg + 复合肥(N15-P15-K15)50kg + 过磷酸钙50kg
- **苗期追肥**：定植后7天，追施尿素5kg/亩，促苗早发
- **初果期**：第一穗果膨大时，追施复合肥20kg/亩
- **盛果期**：每采收2-3次追肥一次，复合肥15kg/亩 + 硫酸钾10kg/亩
- **叶面肥**：花期喷施0.2%硼砂液，膨果期喷施0.3%磷酸二氢钾

#### 黄瓜施肥方案（亩产6000kg目标）
- **基肥**：腐熟有机肥4000-6000kg + 复合肥40kg
- **苗期**：定植后5天，追施稀薄人粪尿或尿素3kg/亩
- **初瓜期**：根瓜坐住后，追施复合肥20kg/亩
- **盛瓜期**：每采收1-2次追肥，复合肥10kg/亩 + 硫酸钾5kg/亩
- **注意**：黄瓜喜肥但不耐浓肥，宜薄肥勤施

#### 辣椒施肥方案
- **基肥**：有机肥2000-3000kg + 复合肥40kg + 硫酸钾15kg
- **苗期**：轻施提苗肥，尿素3-5kg/亩
- **初花期**：控氮增磷钾，追施复合肥15kg/亩
- **盛果期**：每采收一批追肥一次，复合肥15-20kg/亩

### 2.3 有机肥与化肥搭配原则

- 有机肥为主，化肥为辅（有机肥占总养分的50%以上）
- 有机肥作基肥深施，化肥作追肥分次施
- 有机肥改良土壤结构，化肥补充速效养分
- 生物菌肥与有机肥配合使用效果更佳

## 三、灌溉策略

### 3.1 灌溉方式对比

| 方式 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| 滴灌 | 节水50-70%，精准施肥 | 设备成本高，易堵塞 | 设施蔬菜、果园 |
| 喷灌 | 节水30-50%，均匀性好 | 风大时效果差 | 大田作物、草坪 |
| 沟灌 | 成本低，操作简单 | 节水性差，易板结 | 大田作物 |
| 渗灌 | 节水效果最好 | 成本最高，维护复杂 | 高价值设施作物 |

### 3.2 不同作物灌溉要点

#### 番茄灌溉
- **苗期**：适当控水蹲苗，促进根系下扎
- **初花期**：保持土壤相对含水量65-75%
- **膨果期**：需水量大，保持70-80%，但避免忽干忽湿（防裂果）
- **成熟期**：适当控水，提高果实品质
- **关键**：番茄忌大水漫灌，宜小水勤浇

#### 黄瓜灌溉
- **苗期**：少浇勤浇，保持土壤湿润
- **初瓜期**：逐渐增加灌水量
- **盛瓜期**：需水量最大，2-3天浇一次水
- **冬季设施**：晴天上午浇水，避免下午浇水（防夜间湿度过高）

### 3.3 节水灌溉技术

- **土壤湿度传感器**：实时监测土壤含水量，精准灌溉
- **蒸发皿法**：根据蒸发量确定灌溉量
- **作物指标法**：观察叶片萎蔫程度判断灌溉时机
- **覆膜保墒**：地膜覆盖减少蒸发，节水30%以上

## 四、土壤改良

### 4.1 土壤酸碱度调节

| pH范围 | 土壤类型 | 适宜作物 | 调节方法 |
|--------|----------|----------|----------|
| <5.5 | 强酸性 | 蓝莓、杜鹃 | 施用石灰100-200kg/亩 |
| 5.5-6.5 | 微酸性 | 大部分蔬菜 | 适宜范围 |
| 6.5-7.5 | 中性 | 大部分作物 | 最佳范围 |
| 7.5-8.5 | 微碱性 | 棉花、甜菜 | 施用硫磺粉或酸性肥料 |
| >8.5 | 强碱性 | 耐碱作物 | 灌水洗盐+施用石膏 |

### 4.2 有机质提升

- **秸秆还田**：每亩还田300-500kg秸秆，提升有机质0.1-0.2%
- **绿肥种植**：紫云英、苜蓿等翻压入土
- **施用有机肥**：腐熟农家肥2000-3000kg/亩/年
- **生物炭**：施用生物炭改善土壤结构，固碳增肥

### 4.3 盐碱地治理

- **灌水洗盐**：大水漫灌将盐分淋洗至深层
- **排水降盐**：完善排水系统，降低地下水位
- **种植耐盐作物**：棉花、向日葵、甜菜
- **施用改良剂**：石膏、磷石膏、腐殖酸

## 五、设施农业管理

### 5.1 温室环境调控

| 参数 | 白天适宜 | 夜间适宜 | 调控方法 |
|------|----------|----------|----------|
| 温度 | 22-28℃ | 15-18℃ | 通风、加温、遮阳 |
| 湿度 | 60-75% | 70-85% | 通风、滴灌、除湿 |
| CO2浓度 | 600-1000ppm | - | CO2发生器、有机肥分解 |
| 光照 | 30000-50000lux | - | 补光灯、反光膜 |

### 5.2 无土栽培要点

- **营养液配方**：根据作物种类和生长期调整EC值和pH值
- **EC值管理**：苗期1.5-2.0mS/cm，盛果期2.5-3.5mS/cm
- **pH管理**：维持5.5-6.5范围
- **溶氧**：保持营养液溶氧>5mg/L
- **更换频率**：每7-14天更换一次营养液

## 六、常见问题诊断

### 6.1 番茄常见问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 落花落果 | 夜温过高(>20℃)或过低(<15℃) | 调控夜温，使用防落素 |
| 脐腐病 | 缺钙，水分不均 | 补钙+均匀灌溉 |
| 裂果 | 忽干忽湿 | 小水勤浇，保持水分稳定 |
| 空洞果 | 受精不良，氮肥过多 | 适当控氮，合理使用生长调节剂 |
| 着色不良 | 温度不适，光照不足 | 调控温度，改善光照 |

### 6.2 黄瓜常见问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 苦味瓜 | 高温干旱、氮肥过多 | 均匀灌溉，合理施肥 |
| 弯瓜 | 受精不均匀，营养不良 | 人工辅助授粉，补肥 |
| 化瓜 | 夜温高，光照弱 | 降低夜温，疏果 |
| 花打顶 | 低温寡照，根系弱 | 提温，喷施芸苔素 |
