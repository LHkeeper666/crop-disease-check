## 2. 产品简介

### 2.1 系统定位

本系统是面向现代农业的智能化病虫害监测平台，通过整合物联网摄像头、AI图像识别、大数据分析等技术，实现病虫害的自动识别、智能预警、工单管理和防治决策支持。系统支持多租户架构，可同时服务多个农业企业或园区。

### 2.2 核心功能

- **实时监测**：通过RTSP摄像头实时抓拍作物图像，自动识别病虫害
- **AI识别**：基于YOLOv8模型，支持140类病虫害识别（38种病害 + 102种害虫）
- **智能工单**：检测结果自动生成工单，支持状态流转和专家处理
- **数据分析**：农情报表、热力图、七日趋势等多维度数据展示
- **农业大脑**：AI对话助手，支持病虫害咨询和防治建议
- **多租户**：企业级数据隔离，支持邀请码加入机制

### 2.3 应用场景

- 农业园区病虫害监测
- 温室大棚智能管理
- 农技服务远程诊断
- 农业科研数据采集

## 3. 术语与定义

| 术语名称 | 定义说明 |
|----------|----------|
| RTSP | 实时流传输协议（Real Time Streaming Protocol），用于摄像头视频流传输 |
| YOLOv8 | You Only Look Once v8，一种实时目标检测算法，用于病虫害识别 |
| 网格（Grid） | 将监测区域划分为若干网格，每个网格对应一个地块或温室区域 |
| 置信度 | AI识别结果的可信程度，取值0-100%，高于阈值才生成工单 |
| 工单 | 检测到病虫害后自动生成的处理任务，包含位置、类型、严重程度等信息 |
| 多租户 | 系统支持多个企业/组织独立使用，数据相互隔离 |
| WebSocket | 全双工通信协议，用于实时推送检测结果和状态更新 |

## 4. 核心业务流程

### 4.1 流程总览

本系统围绕"监测→识别→预警→处理→分析"的闭环逻辑设计：

摄像头抓拍 → AI识别病虫害 → 置信度判定 → 生成工单 → 专家处理 → 状态流转 → 数据统计 → 日报生成

### 4.2 关键流程说明

#### 4.2.1 实时监测与识别

**角色**：系统自动执行

**核心动作**：
1. 摄像头按设定间隔自动抓拍作物图像
2. 图像上传至推理服务进行AI识别
3. 识别结果包含病虫害类型、置信度、位置信息
4. 置信度高于阈值的结果自动触发工单生成
5. 检测结果通过WebSocket实时推送至前端

#### 4.2.2 工单处理流程

**角色**：操作员、农技专家、基层员工

**核心动作**：
1. 系统自动检测到病虫害后生成工单
2. 工单自动分配给对应区域的负责人
3. 负责人确认工单并开始处理
4. 处理完成后标记工单为已完成
5. 专家可对工单进行诊断和提供建议

#### 4.2.3 数据分析与报告

**角色**：操作员、管理员

**核心动作**：
1. 系统自动汇总每日检测数据
2. 生成农情报表，展示病虫害分布和趋势
3. 支持日报导出为Word格式
4. 热力图展示各区域风险等级

## 5. 系统登录与登出

### 5.1 系统登录

在浏览器地址栏输入系统访问地址进入登录界面，输入正确的用户名和密码后点击登录按钮即可进入系统。系统支持"记住密码"功能，勾选后下次访问将自动填充账号信息。

**默认管理员账号**：admin / admin123

**登录流程**：
1. 打开浏览器，输入系统地址
2. 输入用户名和密码
3. 点击"登录"按钮
4. 首次登录需要通过邮箱验证
5. 登录成功后自动跳转至Dashboard首页

### 5.2 系统登出

点击系统右上角用户头像，在弹出的菜单中选择"退出登录"即可安全退出系统。系统设有会话超时机制，长时间不操作将自动退出。

## 6. 首页（Dashboard）

### 6.1 页面布局

Dashboard首页采用遥测大屏设计，包含以下核心区域：

- **顶部统计卡片**：显示今日检测数、病害数、虫害数、待处理工单数
- **左侧热力图**：展示各网格区域的风险等级，颜色从绿到红表示风险从低到高
- **右侧实时监控**：显示摄像头实时画面和最新检测结果
- **底部趋势图**：展示近7天的病虫害检测趋势

### 6.2 摄像头状态探测（迭代三新增）

**功能说明**：Dashboard加载时自动探测摄像头在线状态，实时更新状态标识。

**操作流程**：
1. 进入Dashboard页面
2. 系统自动执行网络探测，检测摄像头连接状态
3. 摄像头状态图标实时更新：
   - [在线] 绿色：摄像头在线，可正常抓拍
   - [离线] 红色：摄像头离线，无法获取画面
   - [连接中] 黄色：摄像头连接中或信号不稳定
4. 点击摄像头可查看详细信息和触发实时检测

### 6.3 实时检测触发（迭代三新增）

**功能说明**：摄像头在线时可手动触发实时检测，检测结果通过WebSocket实时推送。

**操作流程**：
1. 在Dashboard选择目标摄像头
2. 点击"实时检测"按钮
3. 系统发送检测请求至推理服务
4. 检测进度实时显示
5. 检测完成后结果自动更新到页面
6. 如检测到病虫害，自动生成工单

### 6.4 热力图交互

**功能说明**：热力图展示各网格区域的风险评分，支持拖拽旋转和点击查看详细信息。

**操作流程**：
1. 鼠标拖拽热力图可旋转视角
2. 点击网格区域弹出详细信息：
   - 网格编号和位置
   - 当前风险评分
   - 最近检测记录
   - 关联的温室信息
3. 使用阈值滑块调整风险等级显示范围

## 7. 病虫害信息模块

### 7.1 模块概述

病虫害信息模块提供系统支持的140类病虫害知识库，包括病害图片、名称、描述、防治方法等信息。用户可通过分类浏览或搜索快速查找目标病虫害。

### 7.2 病虫害列表

**功能说明**：展示所有病虫害信息，支持分类筛选和关键词搜索。

**操作流程**：
1. 进入病虫害信息页面
2. 顶部显示分类标签：全部、病害（38类）、虫害（102类）
3. 点击分类标签筛选对应类型
4. 在搜索框输入关键词可快速定位
5. 列表以卡片形式展示，包含图片、名称、类型标签

### 7.3 病虫害描述浮层（迭代三新增）

**功能说明**：鼠标悬停在病虫害图片上时，显示详细描述信息的浮层效果。

**操作流程**：
1. 将鼠标移至病虫害卡片的图片区域
2. 等待0.5秒后，图片上方显示半透明遮罩层
3. 遮罩层展示病虫害描述信息：
   - 病害/虫害名称
   - 主要症状描述
   - 危害程度
   - 常见发生季节
4. 鼠标移出后，遮罩层平滑隐藏
5. 浮层采用毛玻璃效果（backdrop-blur），提升视觉体验

**技术特性**：
- 使用CSS group-hover 实现平滑过渡
- backdrop-blur 毛玻璃效果
- opacity-0 到 opacity-100 的渐变动画
- 描述信息从数据库 description 字段读取

### 7.4 病虫害详情

**功能说明**：查看病虫害的完整信息，包括图片、描述、防治方案等。

**操作流程**：
1. 在病虫害列表中点击目标卡片
2. 进入详情页面，显示完整信息：
   - 高清图片
   - 病害/虫害名称和分类
   - 详细描述（症状、危害、发生规律）
   - 防治方法和推荐药剂
   - 相关案例和参考文献
3. 可通过"返回"按钮回到列表页

## 8. 设备管理

### 8.1 摄像头列表

**功能说明**：管理所有监测摄像头，支持搜索、新增、修改、删除操作。

**操作流程**：
1. 进入设备管理 → 摄像头管理页面
2. 列表展示所有摄像头信息：
   - 摄像头名称
   - RTSP地址
   - 位置坐标
   - 状态（在线/离线/故障）
   - 最后抓拍时间
3. 支持按名称、状态、区域筛选
4. 支持分页浏览，每页20条记录

### 8.2 新增摄像头

**操作流程**：
1. 点击"新增"按钮
2. 填写摄像头信息：
   - 摄像头名称（必填）
   - RTSP主码流地址（必填）
   - RTSP子码流地址（可选）
   - 经度、纬度（可选）
   - 朝向角度（可选）
   - 抓拍分辨率（默认640x640）
   - 抓拍JPEG质量（默认85）
   - 断流重连间隔（默认30秒）
3. 点击"保存"完成新增

### 8.3 修改摄像头

**操作流程**：
1. 在摄像头列表中选择目标记录
2. 点击"修改"按钮
3. 更新摄像头信息
4. 点击"保存"完成修改

### 8.4 删除摄像头

**操作流程**：
1. 在摄像头列表中选择目标记录
2. 点击"删除"按钮
3. 确认删除操作
4. 系统逻辑删除该记录

## 9. 智能工单模块

### 9.1 工单列表

**功能说明**：展示所有工单信息，支持按状态、类型、负责人筛选。

**操作流程**：
1. 进入智能工单页面
2. 列表展示工单信息：
   - 工单编号
   - 病虫害类型（病害/虫害）
   - 检测位置（网格编号）
   - 置信度
   - 状态（待处理/处理中/已完成/忽略）
   - 负责人
   - 创建时间
3. 支持多条件组合筛选
4. 支持分页浏览

### 9.2 工单视图过滤（迭代三新增）

**功能说明**：根据用户角色过滤工单显示内容，员工只能看到分配给自己的工单。

**角色权限说明**：

| 角色 | 可视范围 | 可执行操作 |
|------|----------|------------|
| ADMIN | 所有工单 | 创建、删除、重新分配、修改严重等级、状态流转 |
| EXPERT | 分配给自己的工单 | 确认处理、标记完成、忽略/恢复 |
| MANAGER | 本企业所有工单 | 创建、删除、重新分配、状态流转 |
| STAFF | 分配给自己的工单 | 确认处理、标记完成 |

**操作流程（员工视角）**：
1. 以STAFF角色登录系统
2. 进入智能工单页面
3. 列表仅显示分配给当前用户的工单
4. 可执行操作：
   - 点击"确认处理"开始处理工单
   - 处理完成后点击"标记完成"
   - 无法创建、删除、重新分配工单

**操作流程（专家视角）**：
1. 以EXPERT角色登录系统
2. 进入智能工单页面
3. 列表仅显示分配给当前专家的工单
4. 可执行操作：
   - 点击"确认处理"开始处理工单
   - 处理完成后点击"标记完成"
   - 可选择"忽略"或"恢复"工单
   - 无法创建、删除、重新分配或修改严重等级

### 9.3 工单状态流转

**状态说明**：
- **待处理**：新生成的工单，等待负责人确认
- **处理中**：负责人已确认，正在处理
- **已完成**：处理完成，病虫害已控制
- **忽略**：判定为误报或无需处理

**操作流程**：
1. 在工单列表中选择目标工单
2. 根据当前状态执行相应操作：
   - 待处理 → 点击"确认处理" → 处理中
   - 处理中 → 点击"标记完成" → 已完成
   - 待处理/处理中 → 点击"忽略" → 忽略
   - 忽略 → 点击"恢复" → 待处理
3. 状态变更后系统自动记录操作日志

### 9.4 工单详情

**操作流程**：
1. 在工单列表中点击目标工单
2. 进入详情页面，显示完整信息：
   - 工单基本信息
   - 检测图像和标注结果
   - 病虫害详细信息
   - 处理历史记录
   - 关联的巡检计划
3. 可执行状态流转操作
4. 可查看相关图片（迭代二新增功能）

**操作流程**：

| 实时监控 | 有权限 | 有权限 | 有权限 | 无权限 |
| 农业大脑 | 有权限 | 有权限 | 有权限 | 无权限 |
| 系统管理 | 有权限 | 无权限 | 无权限 | 无权限 |
