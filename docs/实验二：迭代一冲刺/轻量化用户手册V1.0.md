# 一、前言

## 1.1 编写目的

本手册为智慧农业病虫害监测系统迭代一专用操作指南，聚焦迭代一已交付的核心业务功能，帮助用户快速掌握系统核心操作，高效完成病虫害监测与管理。

## 1.2 适用范围

仅适用于智慧农业病虫害监测系统迭代一已交付功能，供系统管理员、农业专家、农场管理者使用。文档不包含系统部署与初始配置，相关需求可联系技术支持部门。

## 1.3 读者对象

- 系统管理员：负责用户管理、系统配置、权限分配
- 农业专家：负责审核上报、制定防治方案、工单管理
- 农场管理者：负责查看报表、监控环境数据、处理工单

由绿盾先锋项目组起草、维护与解释。

# 二、产品简介

## 2.1 系统定位

迭代一聚焦完成前后端联调，实现核心业务功能闭环，支持用户登录注册、环境数据监测、工单管理、图像检测、报表统计、AI对话等核心功能，为后续迭代的高级功能开发奠定基础。

## 2.2 核心功能

- 认证模块：用户登录注册、JWT认证、企业邀请码
- 基础数据：温室管理、环境数据、用户管理
- 工单管理：工单创建、流转、处理闭环
- 图像检测：图片上传、AI病虫害识别、结果入库
- 报表统计：统计概览、日报生成、数据导出
- 智慧大脑：AI农业专家对话、一键生成日报
- 设备管理：摄像头管理、网格区域管理

## 2.3 应用场景

适配温室大棚病虫害监测需求，支持病虫害识别、工单处理、数据统计等核心业务场景，可直接用于农业企业内部管理。

# 三、术语与定义

| 术语名称 | 定义说明 |
|----------|----------|
| JWT | JSON Web Token，用于用户认证和授权的令牌机制 |
| SSE | Server-Sent Events，服务器发送事件，用于流式数据传输 |
| YOLOv8 | 当前行业领先的实时目标检测算法，用于病虫害识别 |
| MINIO | 开源对象存储服务，用于存储图片等非结构化数据 |
| 工单 | 系统自动生成的病虫害处理单，用于跟踪处理进度 |
| 热力图 | 以颜色深浅表示风险等级的可视化图表 |
| 置信度 | AI识别结果的可信程度，以百分比表示 |

# 四、系统登录与登出

## 4.1 系统登录

系统登录是用户访问智慧农业病虫害监测系统的入口。在浏览器地址栏输入系统访问地址（如http://localhost:3000/）进入登录界面，输入正确的用户名和密码后点击登录按钮即可进入系统。为方便下次登录，可在受信任的计算机上勾选"记住密码"选项。

新用户可点击"注册"按钮进行注册，填写用户名、密码、姓名、手机号、邮箱等信息，注册成功后等待管理员审核。注册时可输入企业邀请码加入对应企业。

## 4.2 系统登出

点击系统右上角导航栏头像处的倒三角符号，在弹出的菜单中选择"退出登录"按钮即可安全退出系统。系统设有会话保持时间限制，若长时间不操作系统将自动退出，需重新登录方可使用。

# 五、首页（Dashboard）

登录系统后默认进入Dashboard遥测总览页面，实时展示温室大棚的环境数据和运行状态。

## 5.1 环境数据展示

页面顶部展示实时环境指标：

| 指标 | 说明 |
|------|------|
| 空气温度 | 当前棚内温度（摄氏度） |
| 土壤湿度 | 当前土壤含水量（%） |
| 空气湿度 | 当前棚内相对湿度（%） |
| 光照强度 | 当前光照值（Lux） |
| 能耗数据 | 当前功耗与最大负载 |

## 5.2 温室元数据

可编辑的温室基本信息：

- 区域编号
- 作物种类
- 定植日期
- 地理位置
- 面积

点击"编辑"按钮可修改温室信息。

## 5.3 网格热力图

9宫格网格热力图展示各区域风险状态：

| 颜色 | 状态 | 说明 |
|------|------|------|
| 绿色 | 安全 | 无活跃工单 |
| 蓝色 | 低风险 | 存在低风险工单 |
| 琥珀色 | 中等 | 存在中等风险工单 |
| 橙色 | 高危 | 存在高风险工单 |
| 红色 | 紧急 | 存在紧急工单，闪烁提示 |

点击网格单元格可查看详情。

## 5.4 摄像头状态

展示各摄像头的在线状态和最后抓拍时间。

## 5.5 报警列表

显示当前活跃的报警信息，按严重程度排序。

## 5.6 7日趋势图

展示近7天的病虫害趋势折线图。

# 六、智能工单

## 6.1 工单列表

页面展示所有工单，支持筛选：

- 状态筛选：全部/待处理/处理中/已完成/已忽略
- 严重程度筛选：全部/紧急/高危/中等/低

每个工单卡片显示：
- 工单标题
- 关联网格
- 病虫害名称
- 严重程度标签
- 状态标签
- 置信度

## 6.2 创建工单

1. 点击页面右上角"创建工单"按钮
2. 填写工单信息：
   - 工单标题
   - 关联网格（A1-C3）
   - 病虫害类型（病害/虫害）
   - 病虫害名称
   - 严重程度（紧急/高危/中等/低）
   - 指派人员
   - 检测置信度
3. 点击"创建"按钮

## 6.3 工单详情

点击工单卡片可查看详情：

- 工单基本信息
- 状态流转时间线
- 专家备注
- 回调Token信息

## 6.4 状态流转

工单支持以下状态流转：

待处理 - 处理中 - 已完成
待处理 - 已忽略

点击状态按钮进行流转操作。

## 6.5 严重程度升降级

在工单详情中可调整严重程度：

- 紧急（CRITICAL）：红色闪烁报警
- 高危（HIGH）：橙色报警
- 中等（MEDIUM）：琥珀色提示
- 低（LOW）：蓝色提示

\newpage

# 七、图像检测

## 7.1 图片上传

1. 点击"选择图片"按钮或拖拽图片到上传区域
2. 支持JPG、PNG格式，单张图片大小不超过10MB
3. 支持批量上传，单次最多上传10张
4. 点击"开始检测"按钮

## 7.2 检测结果

检测完成后页面展示：

- 原始图片
- 标注图片（标注病虫害位置）
- 识别结果列表：
  - 病虫害名称
  - 置信度（百分比）
  - 位置坐标

## 7.3 结果入库

检测结果自动保存到系统数据库，可在"我的上报"中查看历史记录。

\newpage

# 八、报表统计

## 8.1 统计概览

页面展示以下统计信息：

- 病害/虫害分布饼图
- 7日趋势折线图
- 统计卡片：
  - 今日上报数
  - 待处理工单数
  - 已处理工单数
  - 高发预警数

支持按时间范围筛选数据。

## 8.2 日报管理

### 8.2.1 日报列表

展示所有日报记录，支持按日期筛选。

### 8.2.2 查看日报

点击日报标题查看详情：

- 日报日期
- 环境数据摘要
- 病虫害统计
- 处理建议

### 8.2.3 生成日报

点击"生成今日日报"按钮，系统自动生成今日日报。

## 8.3 数据导出

1. 选择导出时间范围
2. 选择导出数据类型
3. 点击"导出"按钮
4. 下载Excel格式的数据文件

# 九、智慧大脑

## 9.1 AI对话

1. 在输入框中输入问题
2. 点击"发送"按钮或按回车键
3. AI以流式方式实时回复
4. 支持查看历史对话记录

## 9.2 一键生成日报

点击"一键生成今日日报"按钮，AI自动生成今日农情诊断报告。

## 9.3 会话管理

- 新建会话：点击"新建会话"按钮
- 查看历史：点击历史会话列表
- 删除会话：点击会话右侧删除按钮

# 十一、病虫害图鉴

## 11.1 病虫害图鉴功能

病虫害图鉴功能提供病虫害知识库查询，用户可通过图片或名称查看植物、病虫的详细信息，包括名称、描述、发生条件和防治方法。

## 11.2 病害信息查询

### 11.2.1 病害列表

展示所有病害信息，支持搜索和筛选：

- 按病害名称搜索（支持中英文）
- 按病害类型筛选

每个病害卡片显示：
- 病害名称（中文/英文）
- 病害描述
- 发生条件
- 防治方法

### 11.2.2 查看病害详情

点击病害卡片可查看详情：

- 病害名称（中文/英文）
- 病害描述
- 发生条件
- 防治方法
- 相关图片

## 11.3 虫害信息查询

### 11.3.1 虫害列表

展示所有虫害信息，支持搜索和筛选：

- 按虫害名称搜索
- 按虫害类型筛选

每个虫害卡片显示：
- 虫害名称
- 虫害描述
- 发生条件
- 防治方法

### 11.3.2 查看虫害详情

点击虫害卡片可查看详情：

- 虫害名称
- 虫害描述
- 发生条件
- 防治方法
- 相关图片

## 11.4 图片识别查询

1. 点击"上传图片"按钮或拖拽图片到上传区域
2. 系统自动识别图片中的病虫害
3. 显示识别结果及对应的病虫害信息
4. 支持查看详细信息和防治建议
