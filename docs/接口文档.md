## 1. 通用约定

### 1.1 响应格式

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 1.2 分页请求参数

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| page | int | 否 | 1 | 页码 |
| size | int | 否 | 20 | 每页条数 |
| sort | string | 否 | createdAt,desc | 排序字段,方向 |

### 1.3 分页响应格式

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "records": [],
    "total": 100,
    "page": 1,
    "size": 20,
    "pages": 5
  }
}
```

### 1.4 认证方式

除登录、注册、Token 回调等公开接口外，所有接口需在 Header 中携带：

```
Authorization: Bearer <token>
```

### 1.5 角色权限

| 角色 | 权限说明 |
|------|----------|
| ADMIN | 系统管理员，拥有全部权限 |
| EXPERT | 农技专家，可审核、制定防治方案 |
| MANAGER | 农场管理者，查看数据、报告 |
| VISITOR | 访客，只读权限 |

## 2. 认证模块 (Auth)

### 2.1 用户登录

**POST** `/auth/login`

**请求体：**

```json
{
  "username": "string",
  "password": "string",
  "rememberMe": false
}
```

**响应：**

```json
{
  "code": 200,
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "expiresIn": 86400,
    "userInfo": {
      "id": "uuid",
      "username": "zhangsan",
      "name": "张三",
      "role": "MANAGER",
      "phone": "13800138000"
    }
  }
}
```

**错误码：**

| code | 说明 |
|------|------|
| 40001 | 用户名或密码为空 |
| 40002 | 用户名或密码错误 |
| 40003 | 账号已被禁用 |

---

### 2.2 发送邮箱验证码

**POST** `/auth/send-otp`

**请求体：**

```json
{
  "email": "user@example.com",
  "type": "LOGIN"
}
```

**响应：**

```json
{
  "code": 200,
  "message": "验证码已发送，请查收邮箱"
}
```

**错误码：**

| code | 说明 |
|------|------|
| 40010 | 邮箱格式不正确 |
| 40011 | 验证码发送过于频繁，请1分钟后重试 |

---

### 2.3 邮箱验证码登录

**POST** `/auth/login-by-otp`

**请求体：**

```json
{
  "email": "user@example.com",
  "otp": "123456"
}
```

**响应：** 同 2.1 登录响应

---

### 2.4 退出登录

**POST** `/auth/logout`

**请求头：** `Authorization: Bearer <token>`

**响应：**

```json
{
  "code": 200,
  "message": "退出成功"
}
```

## 6. 图像上报 (Report)

### 6.1 上传图片并上报

**POST** `/report/upload`

**Content-Type:** `multipart/form-data`

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| images | File[] | 是 | 图片文件（最多10张，单张≤10MB，JPG/PNG） |
| gridId | string | 是 | 地块/网格ID |
| cropType | string | 是 | 农作物品种 |
| foundAt | datetime | 是 | 发现时间 |
| description | string | 否 | 补充描述 |

**响应：**

```json
{
  "code": 200,
  "data": {
    "reportId": "uuid",
    "imageUrls": [
      "/images/report/20260609-uuid1.jpg",
      "/images/report/20260609-uuid2.jpg"
    ],
    "status": "PENDING_RECOGNITION"
  },
  "message": "上报成功，正在进行识别"
}
```

**错误码：**

| code | 说明 |
|------|------|
| 40040 | 图片格式不支持（仅支持JPG/PNG） |
| 40041 | 图片大小超过10MB限制 |
| 40042 | 单次上传图片数量超过10张 |
| 40043 | 地块编号或农作物品种为空 |

### 7.3 审核上报

**POST** `/inference/{reportId}/audit`

**权限：** EXPERT, ADMIN

**请求体：**

```json
{
  "action": "APPROVED",
  "comment": ""
}
```

**action 枚举：**
- `APPROVED` - 审核通过
- `REJECTED` - 驳回

**驳回时必须填写 comment（≥10字）：**

```json
{
  "action": "REJECTED",
  "comment": "图片模糊无法识别，建议重新拍摄清晰照片"
}
```

**响应：**

```json
{
  "code": 200,
  "message": "审核成功"
}
```

**错误码：**

| code | 说明 |
|------|------|
| 40050 | 驳回原因不能为空 |
| 40051 | 驳回原因不能少于10字 |
| 40052 | 该记录已被审核 |

---
