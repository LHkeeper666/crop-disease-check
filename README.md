Git 分支命名规范：
- 功能分支：feature/US001-login
- 修复分支：fix/BUG002-query-error
- 版本标签：v1.0.0

文档命名格式：
- 需求文档：XX - 需求规格说明书 V1.0.0.docx
- 设计文档：XX - 详细设计文档 V1.0.0.docx
- 用户手册：XX - 用户手册 V1.0.0.docx
- 版本号规则：采用语义化版本号（主版本。次版本。修订号）
- 文档存储：所有文档统一存储在团队共享空间或代码仓库的 docs 目录
